# 中聯油脂問題油品 · 互動查詢儀表板

依據衛生福利部食品藥物管理署（食藥署）「中聯油脂案專區」**公開公告**整理的互動查詢工具，供民眾與同仁快速比對受影響的商家與商品。

## 線上查詢

👉 **https://macrock23.github.io/zhonglian-oil/**

以瀏覽器（電腦或手機）開啟即可使用，無需安裝或登入。

## 功能

- **我買的商品中了嗎？**：以商品名稱／品牌／類別查詢是否列入終端下架清單（涵蓋 401 項終端商品＋受影響油品），可勾選自己買過的商品成一份退貨核對清單。
- **業者地圖**：360 家下游業者的 2D 互動地圖，可依來源製造商、縣市（可複選）篩選。
- **終端下架完整清單**：官方公告的終端商品完整列表。
- **官方資訊／健康須知**：資料來源、時間、食安專線與健康提醒。
- 支援**明／暗色**切換，設定會記憶在你的瀏覽器。

## 資料來源與時間

- 來源：食藥署「中聯油脂案專區」及相關官方公告。
- 快照時間：**115/7/8（2026-07-08）**。
- 一切以**食藥署最新公告**及商品外包裝實際批號／有效日期為準。

## 免責聲明

本工具為第三方整理，非官方網站、非官方判定，**不構成法律、醫療或投資建議**。商家或品牌被列入官方名單，僅代表其位於問題油品「流向鏈」，**不代表該商家知情，亦不代表其現售商品有害**。如發現錯誤或需更正，請以官方公告為準並洽原業者；食安專線 **1919**。

## 如何更新

資料寫死在 `index.html` 內。要更新時，重新產生新的 `index.html`，於本 repo「Add file → Upload files」上傳覆蓋並 commit，GitHub Pages 幾秒後自動更新。

## 技術說明

- 單一自足的靜態網頁（single-file，純前端，無後端、無資料庫）。
- 地圖使用 Leaflet + MarkerCluster；暗色底圖為 CARTO dark。
- 主題與「我的清單」以瀏覽器 `localStorage` 儲存。
