## 特色

## 四大功能與用法

### 1. 我買的商品中了嗎？（消費者查詢）

判斷你買的商品是否列入預防性下架名單。

- **三種查法**：左側輸入**商品名稱**、點**知名品牌／通路**、或**依類別瀏覽**。
- 中間可切換「**依品項類別／依來源業者**」兩種分類標籤。
- 點任一商品，右側顯示**詳情**（類別、來源業者、縣市、批號、有效日期、狀態、建議動作）。
- **我的清單（退貨核對清單）**：勾選商品前的方框加入清單，可**複製**或**下載 `.txt`**（含標題、產生時間、免責、食安專線）帶去店家核對。
- **最近查詢**：查看過的商品／品牌／類別會記錄在左欄，可點一下重新查詢，也可清除。

### 2. 業者地圖

360 家下游業者的 2D 互動地圖。

- 支援叢集顯示；點標記看業者詳情與涉及品項。
- **篩選**：來源製造商（泰山／福壽／福懋／中聯源頭）、定位狀態（精確／縣市概略）、縣市（**可複選**）、品項。
- 標明**精確定位**與**縣市概略位置**，資訊透明。

### 3. 終端下架完整清單

401 項終端商品的完整表格。

- **點欄位標題可排序**（來源序號／縣市／業者／產品名稱／有效日期）。
- 依**縣市**或**關鍵字**篩選。
- 有效日期為多筆時，收合為「首個日期 等 N 個日期 ▸」，**點開看完整清單**。

### 4. 官方資訊／健康須知

- 事件概要、官方處置與裁罰、苯駢芘（BaP）說明、健康須知（引述專家）。
- 官方連結與**食安專線 1919**。

## 資料範圍與時效

- **來源**：食藥署「中聯油脂案專區」及相關官方公告。
- **範圍**：360 家下游業者、401 項終端商品、30 筆源頭受影響油品。

## 隱私

- 純前端網頁，**沒有後端、沒有資料庫、沒有追蹤**。
- 你的主題、字級、我的清單、最近查詢只存在**你自己的瀏覽器**（`localStorage`），**不會上傳到任何伺服器**。

---

## 無障礙

- 分頁採 `role="tab"` 語意，**鍵盤可聚焦、可用方向鍵切換**。
- 提供 `<nav>` / `<main>` 地標與 `aria-label`；搜尋框有標籤。
- 色彩對比通過 WCAG AA；可**放大字級**輔助閱讀。

---

## 技術架構

- **單一自足的靜態網頁**（single-file，純前端，無後端、無資料庫）。
- 地圖使用 **Leaflet + MarkerCluster**，函式庫**內嵌於頁面**（不依賴外部 CDN）；底圖為 OpenStreetMap／CARTO。
- 資料內嵌於 `index.html`，由 `build_html.py` 從資料檔產生。
- 主題／字級／清單／查詢歷程以瀏覽器 `localStorage` 儲存。
- 部署於 **GitHub Pages**。

---

## 維護與更新

資料寫死在 `index.html` 內，更新步驟：

1. 取得食藥署最新名單，更新資料檔。
2. 執行 `python3 build_html.py` 重新產生 `index.html`。
3. 於本 repo「**Add file → Upload files**」上傳覆蓋 `index.html` 並 commit。
4. GitHub Pages 於數十秒～數分鐘後自動更新。

> 產生器會同時輸出 `index.html` 與 `中聯油脂_下游業者互動地圖.html`（內容相同）。
